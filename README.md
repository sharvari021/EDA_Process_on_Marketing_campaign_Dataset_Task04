# Task 04: Business Insights Report (EDA)

## Objective
Performed Exploratory Data Analysis on a marketing campaign dataset to recommend budget allocation across channels (Facebook Ads, DV360, Google Ads) based on ROI.

## Files
- `Task04_EDA_Marketing.ipynb` — Full analysis notebook
- `Business_Insights_Report.txt` — 1-page business summary
- `chart*.png` — Visualizations (funnel, ROI, CTR, CPC)

## Key Findings
- Best ROI Channel: Google Ads (1104248.2%)
- Best CTR Channel: DV360
- Recommendation: Reallocate budget from Facebook Ads to Google Ads.
  Focus on improving CTR through better ad creatives.
  Use weekday_cat column for time-based optimization.
